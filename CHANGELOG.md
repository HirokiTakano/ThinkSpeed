# Change

このファイルは、ThinkSpeed の変更履歴をまとめるための内部向け記録です。

ユーザーに見せるべき機能追加・仕様変更は、アプリ内サイドバーの「パッチノート」にも掲載します。セキュリティ監査、依存関係監査、内部実装の整理など、ユーザーへの周知が不要な変更はこのファイルにのみ記録します。

## Unreleased

### 内部変更

- セキュリティ監査を実施しました。
- JSON インポートおよび `localStorage` から読み込む Tiptap コンテンツのサニタイズを追加しました。
- バックアップ JSON のインポートに 5 MiB のファイルサイズ上限を追加しました。
- 本番環境の CSP から `unsafe-eval` を削除し、セキュリティヘッダーを強化しました。
- 依存関係監査で確認された Next.js 内部の PostCSS アドバイザリは、安定版 Next.js 側の更新待ちとして継続監視します。

## v0.10 - 2026年5月

### ドラッグ移動

- サイドバーでフォルダをドラッグして並び替えられるようにしました。
- ファイルを同じフォルダ内でドラッグして順番変更できるようにしました。
- ファイルを別フォルダへドラッグして移動できるようにしました。
- ドロップ先が分かりやすいようにハイライト表示を追加しました。

## v0.9 - 2026年5月

### アイコン更新

- アプリ内のブランドアイコンを変更しました。
- ヘッダー、パッチノート、外観、データ管理、カレンダーのアイコン表示を統一しました。

## v0.8 - 2026年4月

### 検索機能

- ノート内検索を追加しました。
- `Ctrl/Cmd + F` またはヘッダーの検索ボタンから検索できるようにしました。
- 「このファイル」「このフォルダ」「すべて」の 3 スコープで検索できるようにしました。
- 複数ヒット時は件数を表示し、それぞれクリックしてジャンプできるようにしました。
- ジャンプ先に黄色ハイライトを表示し、クリックまたは入力で解除できるようにしました。

## v0.7 - 2026年4月

### 文字色カスタマイズ

- エディタの文字色を 3 パターンから選べるようにしました。
- ライトテーマとダークテーマで、それぞれ独立して文字色を設定できるようにしました。

## v0.6 - 2026年4月

### ゴミ箱機能

- 削除したファイル・フォルダをゴミ箱に移動するようにしました。
- ゴミ箱から復元・完全削除できるようにしました。
- ゴミ箱内の項目は 30 日後に自動的に完全削除されるようにしました。

## v0.5 - 2026年4月

### チェックリスト

- チェックリストを追加しました。
- `Ctrl/Cmd + /` でチェックリストを切り替えられるようにしました。
- チェックボックスの見た目を箇条書きと同じスタイルに統一しました。

## v0.4 - 2026年4月

### エクスポート強化

- フォルダ単位・ファイル単位でのエクスポートに対応しました。
- インポート時に同名ファイルを自動リネームし、競合を回避するようにしました。
- エクスポートファイル名を日本語対応の読みやすい形式に変更しました。

## v0.3 - 2025年4月

### 画像サポート

- エディタに画像をそのまま貼り付けられるようにしました。
- `Ctrl/Cmd + V` でスクリーンショットやクリップボードの画像を貼り付けられるようにしました。
- 画像はノートと一緒に `localStorage` に保存されるようにしました。

## v0.2 - 2025年4月

### カレンダー刷新

- カレンダーがノートの内容を自動解析してタスクを登録するように変更しました。
- 「10月4日 13時: タスク名」と書くだけで自動登録できるようにしました。
- 「4/23 15時30分: タスク」形式にも対応しました。
- 全角数字、「終日」タスク、年付き表記にも対応しました。
- カレンダーからタスクをクリックして、該当ノートに即ジャンプできるようにしました。

## v0.1 - 2025年初期

### 初回リリース

- 箇条書きベースのアウトライナーエディタを公開しました。
- フォルダ・ファイルによる階層管理を追加しました。
- `Tab` / `Shift + Tab` による深いインデントに対応しました。
- URL の自動リンク化と YouTube 動画の埋め込みに対応しました。
- ダークモード / ライトモード切替を追加しました。
- 背景色・文字色・アクセントカラーのカスタマイズを追加しました。
- キーボードショートカットのカスタマイズを追加しました。
- JSON でのエクスポート / インポートを追加しました。
- Markdown コピー機能を追加しました。
